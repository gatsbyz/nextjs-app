# Simple Next.js + Tailwind CSS + TypeScript Project

Please use as boilerplate.

## Features

- ⚡️ Next.js 14 with App Router
- ⚛️ React 18
- ✨ TypeScript
- 💨 Tailwind CSS 3 — Configured with CSS Variables to extend the **primary** color
- 🐋 Docker
